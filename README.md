# ⚡ Electronics Learning Projects

A collection of interactive, browser-based tools built for electronics and power electronics students at diploma and engineering level. No installation required — just open the HTML file in any browser.

---

## 🗂️ Project

### Logic Circuit Simulator
**File:** `circuit-simulator.html`

An interactive digital logic simulator for beginners learning Boolean logic and digital electronics.

**Features:**
- Gate Explorer — AND, OR, NOT, NAND, NOR, XOR with live toggle inputs
- Circuit Sandbox — click to place gates, connect with wires, toggle inputs
- Color-coded wires (green = HIGH, gray = LOW)
- Mini challenges to guide learning
- Fully offline — no internet needed

**Topics covered:** Logic gates, truth tables, Boolean algebra, digital circuit design

## 🚀 How to Use

### Option 1 — Run locally (offline)
1. Download the `.html` file of the project you want
2. Double-click to open it in any browser (Chrome recommended)
3. No internet needed (except for the AI Interview tool)

### Option 2 — View live (GitHub Pages)
Visit the live hosted version at:
```
https://<your-username>.github.io/electronics-learning-projects/
```
