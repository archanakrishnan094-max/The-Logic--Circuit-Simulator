# ⚡ Logic Circuit Simulator

An interactive, browser-based logic circuit simulator built for electronics and electrical engineering students. Design digital circuits using standard logic gates and see results in real time — no installation, no setup, just open the HTML file (or visit the live demo) in any browser.

🔗 **Live Demo:** [https://github.com/archanakrishnan094-max]

---

## ✨ Features

- Drag-and-drop style interface to place logic gates (AND, OR, NOT, NAND, NOR, XOR, XNOR)
- Real-time signal simulation — toggle inputs and watch outputs update instantly
- Visual wiring between gates to build custom combinational circuits
- Built entirely with HTML, CSS, and JavaScript — runs fully client-side
- No installation, no dependencies, no backend — works offline once downloaded

---

## 🛠️ Tech Stack

- **HTML5** – structure and layout
- **CSS3** – styling and UI
- **Vanilla JavaScript** – gate logic, simulation engine, and interactivity

---

## 🚀 How to Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/archanakrishnan094-max/The-Logic-Circuit-Simulator.git
   ```
2. Open `logic_circuit_simulator.html` directly in any modern browser (Chrome, Edge, Firefox)
3. Start building circuits — no build step or server required
